# Buenas practicas en Ciberseguridad

1. Crear reglas en firewall para bloquear ciertas urls que sean un peligro para el usuario
2. Utilizar la tecnica de portnocking para desactivar/activar X servicio enviando patrones al servidor.
3. Nunca poner claves API en hardcodeadas en el codigo
4. Crear variables de entorno para luego obtener el valor de la variable de entorno en el codigo
5. Alicar el minimo privilegio a no ser que el usuario requiera ciertos permisos para realizar su trabajo
