# Rustprobe
Rustprobe es un escaner de puertos de alta eficiencia, gran velocidad y de alto rendimiento
desarrollado en rust.


## Objetivos de seguridad

- Garantizar que el escaner respete los estandares de seguridad
- Garantizar que el escaner no colapse los servidores en velocidades minimas
- Garantizar que el escaner no suponga un problema para el equipo
